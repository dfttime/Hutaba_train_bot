# 💪 Workout Tracker Bot

Telegram-бот для записи и анализа тренировок A/B.

## Установка и запуск

```bash
pip install -r requirements.txt
BOT_TOKEN=твой_токен python bot.py
```

Токен получить у [@BotFather](https://t.me/BotFather).

## Команды

| Команда | Действие |
|---|---|
| `/start` | Главное меню |
| `/history` | Быстрый просмотр последних 5 тренировок |
| `/cancel` | Отменить текущую сессию |

## Весь функционал

### 🏋️ Добавить тренировку
- Показывает результаты **последней и позапрошлой** тренировки с объёмом
- Рекомендует какая тренировка (A или B) следующая по очереди
- Пошаговый ввод: бот ведёт по каждому подходу
- Подсказка из прошлой тренировки для каждого упражнения
- Кнопка 🔁 повторить подход / 📋 скопировать всё упражнение из прошлого раза
- Возможность добавить заметку (самочувствие, наблюдения)
- После тренировки — статистика: подходы, повторения, объём, серия

### 📋 История
- Список всех тренировок с датой и объёмом
- Нажми на любую — смотри подробный результат
- Удаление любой тренировки (с подтверждением)

### 📊 Статистика
- 🔥 Текущая и лучшая серия тренировок
- Активность по неделям (текстовый бар-чарт за 4 недели)
- Объём последних тренировок с визуализацией
- 📈 Прогресс по конкретному упражнению (выбор из истории)

### 🏆 Рекорды
- Личный рекорд по максимальному весу для каждого упражнения
- Дата установки рекорда

### ⚖️ Сравнение тренировок
- Выбери любые две тренировки из истории
- Сравнение общего объёма
- Сравнение по каждому упражнению: вес тогда → сейчас с иконками 📈📉➡️

### Управление упражнениями
- ➕ Добавить своё упражнение (сохраняется навсегда)
- 🗑 Убрать упражнение из тренировки
- ♻️ Восстановить убранное упражнение

## Формат ввода подходов

| Ввод | Что означает |
|---|---|
| `60 10` | 60 кг, 10 повторений |
| `60кг 10` / `60x10` | то же самое |
| `10` | только повторения (вес из предыдущего подхода) |
| `без веса 15` / `б/в 15` | без отягощения |
| `50,5 12` | дробный вес |
